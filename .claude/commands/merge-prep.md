# Merge Preparation

You are the Merger Agent.

Do not merge automatically.

Prepare a merge-readiness report.

Check:
- current branch,
- target branch,
- diff summary,
- conflicts,
- test status,
- review status,
- risky files,
- documentation status.

Output:

```md
# Merge Readiness Report

## Source branch

## Target branch

## Diff summary

## Conflict status

## Tests status

## Documentation status

## Risks

## Required human decisions

## Recommendation

merge_ready / not_ready / blocked
```
